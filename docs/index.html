Fire severity analysis for South Australia

Work in Progress

This is currently a prototype using dummy data for demonstration purposes. The interactive map below shows simulated fire severity analysis results.

About This Project

This tool analyzes fire severity using the differenced Normalized Burn Ratio (dNBR) method for interactive mapping and analysis. It processes Area of Interest (AOI) data and generates interactive maps showing fire severity levels across South Australia.
