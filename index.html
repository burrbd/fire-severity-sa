Interactive mapping and analysis of fire severity using dNBR calculations

About This Project

This tool analyzes fire severity using the differenced Normalized Burn Ratio (dNBR) method. It processes Area of Interest (AOI) data and generates interactive maps showing fire severity levels.

How to Use

  1. Prepare your AOI data in GeoJSON format
  2. Run the analysis pipeline: python src/process_aoi.py your_aoi.geojson
  3. View the generated interactive map below

Interactive Fire Severity Map

Legend

  • Green - Low Severity (0-1)
  • Orange - Medium Severity (1-3)
  • Red - High Severity (3-4)
